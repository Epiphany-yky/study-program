# C++ 高并发TCP/HTTP服务器

## 项目简介
本项目是一个基于C++11实现的**高并发多线程HTTP服务器**，从零实现了线程池、Socket网络编程、HTTP协议等核心功能，
支持多客户端同时访问，可直接部署运行，符合计算机网络、操作系统课程的课程设计要求。

## 核心功能
1. **高并发多线程处理**：基于自定义线程池，支持多客户端同时连接，避免单线程阻塞
2. **标准HTTP协议支持**：兼容HTTP 1.1，可被浏览器、curl等标准客户端正常访问
3. **优雅退出机制**：支持`Ctrl+C`安全关闭，避免资源泄漏
4. **完善日志系统**：详细记录运行状态、客户端连接、请求处理，便于调试
5. **可配置线程数**：可根据性能调整线程池大小，适配不同并发需求
6. **端口复用**：支持端口快速复用，避免重启时端口占用

## 核心原理
### 线程池实现
任务队列：线程安全的任务队列，由互斥锁保护
条件变量：实现线程阻塞 / 唤醒，无任务时休眠，避免 CPU 空转
线程复用：预先创建线程，避免频繁创建 / 销毁的性能开销
原子变量：控制线程池停止，保证多线程内存可见性
### Socket 通信流程
plaintext
socket() → setsockopt() → bind() → listen() → pselect() → accept() → read() → write() → close()
pselect实现可中断监听，支持优雅退出
每个客户端连接由独立线程处理，实现并发
### HTTP 协议处理
接收客户端 HTTP 请求，构造标准 HTTP 响应头
符合 HTTP 1.1 规范，支持短连接模式
